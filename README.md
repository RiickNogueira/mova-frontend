| Requisito | Link para download                                                  |
| --------- | ------------------------------------------------------------------- |
| Node      | https://nodejs.org/en/download/ |

## Instalação e execução

Faça download do projeto e execute os seguintes comandos na raiz

```sh
npm i
npm run serve
```

_A API rodará na porta 8080. http://localhost:8080_
